Cosmetic Insights Dashboard

Analyzing Cosmetics Trends and Consumer Insights using Tableau

Project Overview

This project provides insights into cosmetic products by analyzing pricing, premium brands, product categories, brand popularity, price distribution, and skin type compatibility using Tableau.

Compare average prices across cosmetic brands.

Top Premium Brands

Identify luxury cosmetic brands based on pricing.

Product Categories

Analyze the distribution of product types.

Price Distribution

Understand pricing trends using histogram analysis.

Skin Type Compatibility

Compare products suitable for different skin types.

Brand-wise Product Count

Discover the brands with the highest number of products.

Interactive Tableau Dashboard

Developed using Flask, HTML, CSS, JavaScript and Tableau Public.
