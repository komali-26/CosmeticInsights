# Cosmetic Insights: Navigating Cosmetics Trends and Consumer Insights with Tableau

Project Overview
This project analyzes cosmetic products using **Tableau** and **Flask** to provide meaningful insights into the cosmetics industry. The dashboard helps users understand pricing trends, premium brands, product categories, and skin type compatibility through interactive visualizations.

Features
- Brand-wise Product Count
- Average Price by Brand
- Top Premium Brands by Average Price
- Product Category Distribution
- Price Distribution (Histogram)
- Skin Type Compatibility
- Interactive Tableau Dashboard
- Flask-based Web Application

Technologies Used
- Python
- Flask
- Tableau Public
- HTML5
- CSS3
- JavaScript
- Git & GitHub

 How to Run the Project

 1. Clone the Repository
 bash

2. Open the Project
bash
cd CosmeticInsights
3. Install Dependencies

bash
pip install -r requirements.txt
4. Run the Flask Application
  bash
  python app.py
5. Open the Browser
  http://127.0.0.1:5000
